Before the demo

Common questions

How long is the demo?
30 minutes. We spend the first 5 minutes understanding your context, then run the pipeline live for 20 minutes, then take questions. If you want to go deeper, we can book a follow-up technical session.
Do I need to prepare anything?
Nothing. We run the demo on our own codebase. If you want to see it run against your own code, let us know in the request and we'll arrange a technical pre-call to set it up.
Is this a sales call?
No. It's a working demo. We show you the tool running — not slides about the tool. Pricing is on the website; we don't hide it. If there's a fit after the demo, we'll say so.
Do I need to sign an NDA first?
Not for the demo itself — everything shown is at Public Safe level. If you want to see investor materials or deeper technical documentation, we'll send an NDA first.
What if I just want to discuss our compliance situation first?
That's fine too — email us at [email protected] with a brief description of your use case and we'll have a plain conversation before scheduling anything formal.
